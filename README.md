Description

The Blinky Board is a basic electronics project where an LED blinks continuously. It is designed to show how simple timing circuits work.
The circuit uses a NE555 Timer IC in astable mode to generate a repeating signal. This signal turns the LED on and off, which creates the blinking effect. The speed of blinking depends on the values of the resistors and capacitor used in the circuit.

Purpose

I made this project to learn the basics of electronic circuits, especially how timing and oscillation work. It also helped me understand soldering and how to design a simple PCB.

Usage Instructions

Connect the circuit to a power supply between 5V and 9V. Once powered, the LED will start blinking automatically. You can change the blinking speed by adjusting the resistor or capacitor values.

Build Of Materials

| Component                | Quantity | Description                    |
| ------------------------ | -------- | ------------------------------ |
| NE555 Timer IC           | 1        | Generates clock signal         |
| CD4017 Decade Counter IC | 1        | Controls LED sequence          |
| Power Header (VCC/GND)   | 1        | Power input                    |
| Debug Header             | 1        | For testing                    |
| Electrolytic Capacitor   | 1        | Timing component (directional) |
| Ceramic Capacitor        | 1        | Stabilizes circuit             |
| 1kΩ Resistor             | 1        | Timing/current control         |
| 470Ω Resistor            | 10       | Limits LED current             |
| Potentiometer            | 1        | Adjusts speed                  |
| LED                      | 10       | Output lights                  |
